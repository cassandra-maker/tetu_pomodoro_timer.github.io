# Focus Flow - Web Audio Pomodoro

ホワイトノイズ生成機能を搭載した、超軽量・データ通信不要のポモドーロタイマーです。
PCおよびAndroidでの利用を最適化しており、PWAとしてホーム画面にインストール可能です。



## 🌟 主な特徴

- **ホワイトノイズ自動生成**: Web Audio APIを使用し、ブラウザ上でリアルタイムに音を合成。音源ファイルのダウンロードが不要です。
- **データ通信ゼロ (Zero Data Usage)**: 一度読み込めば、以降はオフラインで動作。パケット消費を気にせず集中できます。
- **バイブレーション通知**: 集中終了時と休憩終了時で異なるパターンで振動し、画面を見なくても状況を把握できます。
- **モダンなUI**: ニューモーフィズムを採用した、目に優しく落ち着いたデザイン。
- **PWA対応**: Androidのホーム画面にアプリとして追加でき、フルスクリーンで動作します。

## 📊 通信量と効率性の比較

このアプリは、音源を「再生」するのではなく、プログラムで「生成」するため、圧倒的にエコです。

| 比較項目 | 一般的な動画サイト <br>(YouTube等) | 本アプリ  | 有用性のポイント |
| :--- | :--- | :--- | :--- |
| **初期読み込み量** | 数MB 〜 数十MB | **約 0.1MB 以下** | テキストデータのみなので一瞬で起動 |
| **25分間の通信量** | 約 200MB 〜 500MB | **0 バイト** | 内部で音を合成するため通信が発生しない |
| **2回目以降の起動** | 毎回通信が発生する | **0 バイト (完全オフライン可)** | Service Workerにより端末内にキャッシュ |
| **バッテリー消費** | 通信と描画で負荷が高い | **極めて低い** | 無駄な通信モジュールを動かしません |



## 🚀 使い方

1. **スタート**: 「START」ボタンを押すと、25分のカウントダウンとホワイトノイズが始まります。
2. **一時停止**: 作業中に「PAUSE」を押すと、タイマーと音が停止します。
3. **通知**: 時間が終了すると、アラートとともにスマホが振動します。
  スマホ本体が「サイレントモード（バイブなし）」になっていると、
  ブラウザからのバイブレーションが無視されます。
  使用する際は、**「バイブレーションありのマナーモード」**になっているか確認してください。
4. **リセット**: 「RESET」ボタンで最初の状態に戻ります。

### Androidでのインストール方法
1. Chromeブラウザで公開したURLを開きます。
2. 右上のメニュー（⋮）から「ホーム画面に追加」または「アプリをインストール」を選択します。
3. 以降はホーム画面のアイコンから、オフラインでも利用可能です。
