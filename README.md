# tetu_pomodoro_timer.github.io

**スマホで使えるオフライン対応（初回＆キャッシュ消えた場合のみ通信）のポモドーロタイマーです。**
* ホワイトノイズの自動生成: Web Audio API を使い、数学的にノイズを作成。
  音源ファイルのダウンロードが不要で、ループの継ぎ目もありません。
* バイブレーション通知: 集中終了時は「長く3回」、休憩終了時は「短く2回」スマホが震えます。
* 通信料ゼロ: 最初の読み込み以降は、sw.js によりオフラインで動作。データ通信は発生しません。
* モダンなUI: ニューモーフィズムを採用し、目に優しい配色と直感的な円形進捗バーを搭載。

マナーモード設定: スマホ本体が「サイレントモード（バイブなし）」になっていると、
ブラウザからのバイブレーションが無視されます。
使用する際は、**「バイブレーションありのマナーモード」**になっているか確認してください。
